Feedstock for [eic-smear](https://github.com/eic/eic-smear) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Fast detector smearing for EIC physics studies (ROOT-based)

## Install

```bash
conda install -c hep-forge -c conda-forge eic-smear
```
